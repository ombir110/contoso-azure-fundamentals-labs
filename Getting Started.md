# Getting Started

## Accessing Azure Portal

1. Open a browser and navigate to:
   https://portal.azure.com
2. Sign in using your Azure credentials

## Azure Portal Overview

The Azure Portal is a web-based interface used to create and manage resources.

- Use the **Search bar** to quickly find services
- Use **Create a resource** to deploy services
- Use **Resource Groups** to organize resources

## Key Concepts

- **Resource**: An instance of a service (VM, Storage, etc.)
- **Resource Group**: Logical container for resources
- **Subscription**: Billing and access boundary
