# Overview

This lab is designed to introduce participants to core services of Microsoft Azure Cloud Platform. 

In this workshop, you will gain hands-on experience in creating and managing Azure resources through the Azure Portal.

## 🎯 Learning Objectives

By completing this lab, you will be able to:

- Create and manage Resource Groups
- Deploy and connect to Virtual Machines
- Build and access Web Apps
- Create and use Storage Accounts
- Monitor resources using Azure Monitor
- Perform cleanup to avoid unnecessary costs

## 🧪 Lab Exercises

1. Resource Group  
2. Virtual Machine  
3. Web App  
4. Storage Account  
5. Monitoring  
6. Cleanup  

## ⏱️ Estimated Duration

60–90 minutes
