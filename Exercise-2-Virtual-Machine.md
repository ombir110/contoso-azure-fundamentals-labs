# 🖥️ Exercise 2: Virtual Machine Deployment

## ⏱️ Duration: 20 minutes

## 🔹 Task 1: Create VM

### Steps
1. Go to Virtual Machines
2. Click **Create**

### Configuration
- Name: contoso-vm
- Image: Windows Server 2022
- Size: B1s

Allow RDP (3389)

---

## 🔹 Task 2: Connect to VM

### Steps
1. Go to VM → Connect
2. Use Bastion or browser RDP
