# Exercise 2: Create a Virtual Machine

## 🎯 Objective

Deploy and connect to a Virtual Machine.

---

## Steps

### Step 1: Start VM Creation

- Click **Create a resource**
- Select **Virtual Machine**

  <div style="text-align: center;">
  <img src="images/5.jpg" width="600"/>
  <p><em>Virtual Machine Creation Page</em></p>
  </div>  

---

### Step 2: Configure Basics

- Resource Group: `testdemo`
- VM Name: `testdemoVm`
- Region: Same as RG
- Image: Windows Server / Ubuntu
- Username and Password

  <div style="text-align: center;">
  <img src="images/6.jpg" width="600"/>
  <p><em>VM basic configuration</em></p>
  </div>  
   

---

### Step 3: Select Size

- Choose **Standard B1s**

  <div style="text-align: center;">
  <img src="images/7.jpg" width="600"/>
  <p><em>VM size selection</em></p>
  </div>    

---

### Step 4: Configure Networking

- Public IP: Enabled  
- Allow inbound ports: RDP (3389) / SSH (22)

  <div style="text-align: center;">
  <img src="images/8.jpg" width="600"/>
  <p><em>Networking tab</em></p>
  </div>  

---

### Step 5: Review and Create

- Click **Review + Create**
- Click **Create**

  <div style="text-align: center;">
  <img src="images/9.jpg" width="600"/>
  <p><em>Review page</em></p>
  </div>  

---

### Step 6: Deployment Completion

- Wait for deployment
- Click **Go to resource**

  <div style="text-align: center;">
  <img src="images/10.jpg" width="600"/>
  <p><em>Deployment Success</em></p>
  </div>
