# Exercise 3: Create a Web App

## 🎯 Objective

Deploy a web application using Azure App Service.

---

## Steps

### Step 1: Create Web App

- Click **Create a resource**
- Select **Web App**

  <div style="text-align: center;">
  <img src="images/12.jpg" width="600"/>
  <p><em>Web App Creation page</em></p>
  </div>    

---

### Step 2: Configure Basics

- Resource Group: `testdemo`
- App Name: Unique name ex:`testdemo5123`
- Runtime Stack: Node / .NET / PHP

  <div style="text-align: center;">
  <img src="images/13.jpg" width="600"/>
  <p><em>Web App Basics tab</em></p>
  </div>    

---

### Step 3: Configure App Service Plan

- Select Free Tier (F1)

  <div style="text-align: center;">
  <img src="images/14.jpg" width="600"/>
  <p><em>Web App Basics Service plan</em></p>
  </div>
---

### Step 4: Review and Create

- Click **Create**

  <div style="text-align: center;">
  <img src="images/15.jpg" width="600"/>
  <p><em>Review page</em></p>
  </div>

### Step 5: Access Web App
